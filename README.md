## 🚀 Cài đặt và chạy

### Yêu cầu hệ thống

- Flutter SDK >= 3.0.0
- Dart SDK >= 3.0.0
- Android Studio / VS Code
- Android SDK hoặc iOS SDK

### Cài đặt

```bash
# Clone repository
git clone https://github.com/EXE101-CareNest/mobile.git
cd mobile

# Cài đặt dependencies
flutter pub get

# Chạy ứng dụng
flutter run
```

## 📁 Cấu trúc project

```
lib/
```
